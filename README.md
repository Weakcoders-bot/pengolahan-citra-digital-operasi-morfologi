# Operasi Morfologi pada Citra Biner

## Mata Kuliah
Pengolahan Citra Digital  
Universitas Pamulang

## Deskripsi
Project ini membahas implementasi operasi morfologi pada citra biner menggunakan Python dan OpenCV.
Operasi yang diterapkan meliputi erosion, dilation, opening, closing, morphological gradient,
top hat, dan black hat.

## Tools & Library
- Python 3
- OpenCV (cv2)
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset / Citra
- Logo Universitas Pamulang (PNG)

## Tahapan Pengolahan
1. Load citra
2. Konversi ke grayscale
3. Thresholding (citra biner)
4. Structuring Element
5. Operasi Morfologi:
   - Erosion
   - Dilation
   - Opening
   - Closing
   - Morphological Gradient
   - Top Hat & Black Hat

## Cara Menjalankan
1. Buka folder `notebook/`
2. Jalankan file `OPERASI_MORFOLOGI_PADA_CITRA_BINER.ipynb`
3. Run cell dari atas ke bawah
